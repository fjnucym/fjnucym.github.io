窗口部件

基础窗口部件QWidget

QWidget 类是所有用户界面对象的基类,被成为基础窗口部件。QWidget继承自QObject类是所有支持Qt对象模型(Qt Object Model)的基类,QPaintDevice类是所有可以绘制的对象的基类。

新建QWidget窗口&向窗口添加子部件

#include <QWidget>
#include <QLabel>

int main()
{
 // 新建QWidegt类对象,默认parent参数是0(不传default=0),所以它是独立的 窗口(它是窗口)
	QWidget *widget = new QWidget();
 // 新建QLable对象,默认parent参数是0,所以它仍然被当作单独窗口处理
 QLabel *label1 = new QLabel();
 // 新建QLable对象,默认parent参数是widget,即它是widget的子部件
 QWidget *label2 = new QWidget(widget);
}

这里注意部件创建时如果需要它作为子部件,则我们得传入窗口的指针！

窗口部件的定义

窗口部件(Widget)这里简称部件,是Qt中建立用户界面的主要元素。像主窗口、对话框、标签、按钮、文本输入框等都是窗口部件。这些部件可以接收用户输入,显示数据和状态信息,并且可以在屏幕上绘制自己,有些也可以作为一个容器来放置其他部件

窗口类型

一个部件在创建时可以传入参数

  • 第一个参数是父类的指针(作为父窗口部件,传0代表独立)

  • 第二个参数是一个Qt::WindowFlags枚举类型,代表 窗口类型(WindowType)或窗口标志(WindowFlags).

    前者可以定义窗口类型,如传入Qt::WindowFlags f = 0(不传就是默认0),还有很多别的类型如Qt::Dialog、Qt::SpashScreen，表示对话框和欢迎窗口

    参数示例

QWidget *widget = new QWidget(0, Qt::Dialog);

后者可以定义窗口标志(可以和窗口类型一起使用位或传入),作用是更改窗口的标题栏和边框

// 无边框窗口
QWidget *widget1 = new QWidget(0, Qt::Dialog|Qt::FramelessWindowHint);
// 可以使得该部件停留在其他部件上面
QWidget *label = new QWidget(0, Qt::SplashScreen|Qt::WindowStaysOnTopHint);

这里只是举例,其实还有很多类型和标志

详细请前往Qt Examples and Demos工具里的Widgets分类中查看 | 或者可以去欢迎界面看示例
